================================================================================
DESIGN PRINCIPLES:
================================================================================

1. Minimalist & Clean: Remove clutter, focus on content
2. Dark Theme: Modern, easy on eyes, professional appearance
3. Responsive: Mobile-first design approach
4. Accessible: Keyboard navigation, semantic HTML, ARIA labels
5. Performance: Fast loading, optimized assets
6. Typography-Focused: Clean fonts, good hierarchy
7. Subtle Animations: Smooth transitions without distracting
8. Micro-Interactions: Hover states, active states, feedback
9. Consistency: Unified color scheme, spacing, sizing
10. Accessibility First: High contrast, readable text, accessible navigation

================================================================================
DEVELOPMENT NOTES:
================================================================================

- Use semantic HTML5 elements
- CSS Grid or Flexbox for layouts
- CSS variables for theming and consistency
- Mobile-first CSS approach (base styles mobile, then media queries for larger screens)
- Vanilla JavaScript (no framework required for static HTML version)
- Google Fonts for typography
- Inline SVG for icons (no external icon library needed)
- Optimize images (use SVG where possible, compress JPG/PNG)
- Test on multiple browsers and devices
- Validate HTML and CSS
- Check accessibility with WAVE or similar tools
- Implement proper SEO meta tags
- Use CSS transitions for smooth animations
