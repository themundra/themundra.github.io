Trackify

Personal Finance Tracker • Privacy-First • Flutter

Trackify is a privacy-first personal finance tracker I built with Flutter. It features encrypted local storage, optional Firebase sync, and comprehensive analytics for tracking weekly trends and category breakdowns. The app prioritizes user data security while providing a delightful user experience.

Key Features

  • Encrypted Local Storage: All data is encrypted locally on your device
  • Optional Cloud Sync: Securely sync your data to Firestore
  • Multi-Method Tracking: Track expenses across multiple payment methods
  • Responsive Charts: Beautiful, animated charts for expense visualization
  • Weekly Analytics: Comprehensive trend analysis and category breakdowns
  • Accessible Interface: Built with accessibility best practices

Project Details

  • Role: Primary developer — app architecture, storage layer, analytics engine
  • Status: Closed-source project; APK available on request
  • Tech Stack: Flutter, Dart, Hive (local storage), Firebase (optional sync)
